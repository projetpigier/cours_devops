# Cours DevOps avec Java

Dépôt du cours **DevOps avec Java** — IUT / Licence Pro.

## Structure du cours

| Chapitre | Titre | Type |
|---|---|---|
| `chapitre_01/` | Concepts Fondamentaux de DevOps | Cours |
| `chapitre_02/` | Présentation de l'Écosystème Java | Cours |
| `chapitre_03/` | Gestion du Code avec Git | TD |
| `chapitre_04/` | Lancement d'une Application avec Docker | TD |
| `chapitre_05/` | Programmation Impérative en Java | Cours |
| `chapitre_06/` | Programmation Orientée Objet en Java | Cours |
| `chapitre_07/` | Automatisation des Tâches avec Jenkins | TD |
| `chapitre_08/` | Tests Unitaires | TD |
| `chapitre_09/` | Utilisation des Composants Graphiques | Cours |
| `chapitre_10/` | Intégration des Bases de Données | Cours |
| `chapitre_11/` | Développement d'une Application Windows | TD |
| `chapitre_12/` | Intégration d'une Base de Données MySQL | TD |

## Prérequis

- Un compte **GitHub** gratuit
- Un navigateur web récent (Chrome, Firefox, Edge)

> Le cours se suit entièrement dans **GitHub Codespaces** : aucune installation locale n'est requise.

## Technologies utilisées

| Technologie | Rôle |
|---|---|
| Java 21 | Langage de développement principal |
| Maven | Build et gestion des dépendances |
| Lombok | Réduction du code répétitif |
| Java Swing | Interface graphique de l'application |
| JDBC + MySQL | Accès à la base de données |
| Docker + Docker Compose | Conteneurisation et orchestration |
| Git + GitHub | Gestion de version et collaboration |
| Jenkins | Pipeline CI/CD |
| JUnit 5 | Tests unitaires |
| GitHub Codespaces | Environnement de développement cloud |

## Pour commencer

1. **Forkez** ce dépôt (bouton `Fork` en haut à droite sur GitHub)
2. **Ouvrez un Codespace** depuis votre fork (`Code` → `Codespaces` → `Create codespace on main`)
3. Consultez le `README.md` de chaque chapitre pour les consignes détaillées
